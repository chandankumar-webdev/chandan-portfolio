# Portfolio

Personal portfolio built with **Next.js (App Router)**, **React**, and **Tailwind CSS**.

## Commands

```bash
npm install
npm run dev    # http://localhost:3000
npm run build
npm run start  # production server after build
```

## Content

Edit `src/data/site.ts` for links (LinkedIn, GitHub), experience, projects, and courses.

## Deploy

- **Vercel**: connect the repo — Next.js is detected automatically.
- **Static export** (optional): add `output: "export"` to `next.config.ts` if you only need static files on GitHub Pages.
